# MY-MARKET-APP
Для запуска приложения:
1. Измените настройки для подключения к БД в файле .env
2. Запустить Docker, т.к он потребуется для тестов приложения.
3. Выполните команду для сборки приложения mvn clean install
4. Выполнить команду docker compose up
5. Тестовые пользователи ддя проверки авторизации
 - admin:password
 - user:password
